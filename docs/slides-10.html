Onde NÃO estão as pontes

Busca por obras-ponte via betweenness centrality + null configuration (1000 sorteios):

  • Nenhuma obra atinge significância como conector cross-axis no esqueleto par-a-par.
  • 26 obras tocam ≥2 eixos por vocabulário, mas estruturalmente são satélites.
  • O par-a-par mostra silos rígidos. A convergência não está aqui.

No hipergrafo, mesmo as 482 hiperarestas trans-eixo realizadas (de 1341) ainda estão abaixo do que o acaso produziria — sinal de silo é robusto na ordem superior.
