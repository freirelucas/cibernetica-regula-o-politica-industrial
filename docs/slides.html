A pergunta

Três tradições compartilham um problema comum, mas se comunicam pouco:

  • Cibernética organizacional — controle e viabilidade de sistemas complexos (Beer, Ashby)
  • Instrumentos de governo — meios pelos quais o estado age (Hood, Margetts)
  • Política industrial — ação estatal deliberada sobre a estrutura produtiva (Rodrik, Mazzucato)

O campo intelectual já os conecta? Onde? Quanto? Por quê não?

O método — um funil de evidência

  • 13 obras-semente canônicas (10 + 3 de Lange para resgatar a cibernética econômica)
  • Bola de neve OpenAlex (progressiva + regressiva) → corpus de ~800 obras
  • Redes de cocitação + acoplamento + hipergrafo (XGI)
  • Comunidades Leiden, intermediação, rajadas Kleinberg, coeficiente de beleza
  • Modelos nulos: configuration null pairwise + stub-shuffle hipergrafo
  • Coautoria completa (817 autores) + Gould-Fernandez brokerage
  • Saída operacional: lista para Rayyan com 14+ tags multinível

Pontes textuais — rejeitadas

A busca por obras-ponte (conectores significativos entre eixos) via betweenness centrality:

O eco do Cybersyn

A cristalização do silo nos 1980s coincide com a ruptura institucional dos programas que poderiam tê-los unido:

  • Cybersyn (1971-73): único experimento institucional de cibernética aplicada a política industrial → destruído pelo golpe
  • Beer publica Heart of Enterprise (1979) — momento de auge editorial → recepção compartimentada
  • Hood publica Tools of Government (1983) — fundação do campo de instrumentos → sem diálogo com Beer

Os silos atuais são herdeiros de uma fragmentação histórica reparável.

Brokerage roles (Gould-Fernandez)

Cada autor num triplete A-B-C ocupa um papel:

  • coordinator — todos no mesmo grupo (intra-axis)
  • gatekeeper — traz info para o grupo
  • representative — leva info do grupo para fora
  • liaison — entre dois grupos externos ao seu
  • itinerant — atravessa três grupos diferentes

Distribuição: 292 coordinator (intra) · 26 gatekeeper · 25 representative · 4 itinerant · 2 liaison.

Mesmo no plano social, os cross-axis brokerage roles são raros — confirma silos.

Implicações para a política industrial

Para a NIB 2023 e o trabalho do IPEA:

  • Não esperar emergência da convergência: o silo é histórico, robusto, multinível
  • Curadoria intencional de listas, cursos, edições comentadas — não workshop
  • Tradutores institucionais reais existem (Nuti, Espejo, Peter John, Boulding) — partir deles
  • O IPEA pode ser broker: instituição que opera nos 3 eixos simultaneamente é rara
  • O Cybersyn como referência histórica viva — não como nostalgia, mas como protocolo

Próximos passos

  • Estender classificador para OpenAlex concepts (captura temática além do título)
  • Incluir tradições adjacentes como sementes (Polanyi, Hirschman, complexity)
  • SR efetiva no Rayyan pelos 3 pesquisadores → publicação
  • Paper teórico: "Política industrial como sistema viável" (Beer × Mazzucato × Hood)
  • Aplicação ao desenho do NIB 2023 — concretização prática

Honestidade epistêmica — o que NÃO mostramos

  • OpenAlex não cobre livros sistematicamente → bola de neve regressiva é parcial
  • Vocabulário em título é conservador — abstracts/concepts não usados ainda
  • Modelos nulos para hipergrafo são heurísticos (stub-shuffle); literatura ainda nova
  • Cross-axis em autor tem sample-size pequeno para vários autores
  • Brasil tem busca documentada mas sem protocolo PRISMA completo
  • Sem triangulação multi-database (Scopus, WoS)

Limites declarados → leituras honestas.
